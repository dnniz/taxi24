
## Description

[Nest](https://github.com/nestjs/nest) framework TypeScript starter repository.


## Preconditions

DATABASE: postgresql version 12 min
POSTGIS: installation https://postgis.net/workshops/postgis-intro/installation.html

script execution:
SELECT postgis_full_version();
CREATE EXTENSION IF NOT EXISTS postgis;


## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
